# Contributing

## Pull Request 전 확인

- [ ] 브랜치 목적이 하나인가?
- [ ] 관련 없는 파일을 수정하지 않았는가?
- [ ] 코어 동결 영역을 건드리지 않았는가?
- [ ] 테스트 체크리스트를 확인했는가?
- [ ] 문서를 함께 업데이트했는가?

## PR 제목 예시

- `fix: hide inbound map placeholder flash`
- `ui: adjust sidebar and top spacing`
- `feat: add product image rotation controls`
- `refactor: extract inventory transaction helpers`
